My Work

What I Do

I am a user focused software engineer that has an obsession with efficient design. I use Web 2.0 technologies to build innovative new ways to experience and interact with digital media. I thrive in creative, cutting edge, and fast paced environments.


I am currently working on a sharing economy app to utilize unused items laying around people's houses. A while back, while I was staying in an AirBnB in Hawaii attempting to learn to surf, there was an unused surfboard hanging on the wall. It seemed to me that there must be thousands of items like this in people's houses across the island and that it would be much more efficient to put them to use. So, I started building an app to make this easy to do. I am building the app end to end in Javascript, using MongoDB, ExpressJS, Angular2, and NodeJS and the Ionic Framework to deploy Angular code natively to mobile devices.
